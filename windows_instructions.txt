Instructions to execute the program in Windows:
- Using command-line:
    - Open anaconda-prompt
    - Move to the project folder (molecular-dynamics-simulator) using "cd" command
    - Type: 
        - python run_simulation.py --data_file='route_to_data_file.txt' to read a data file. A data file named data.txt is available
        - python run_simulation.py to deploy the GUI
- Using executable file: double click on the executable_for_windows.exe file. Changing the files in the project won't affect the execution with this method. A new executable must be created in order to see the changes. IMPORTANT: the executable file must remain in the position where it is provided
